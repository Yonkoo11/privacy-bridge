Protocol Flow
01 // Lock
Deposit a fixed denomination into the PrivacyBridge contract. A Poseidon commitment is inserted into the on-chain Merkle tree.
Flow EVM
02 // Prove
Generate a Groth16 proof off-chain. The proof attests to knowledge of a valid commitment in the tree without revealing which one.
Off-chain / Browser
03 // Verify
The garaga verifier on Starknet checks the Groth16 proof on-chain. Nullifier hash is checked against the spent set.
Starknet
04 // Claim
Tokens are minted to the recipient on Starknet. The nullifier is marked spent. No link to the original deposit is visible.
